## Setup

1. Clone the repository:
    ```bash
    git clone https://github.com/your-username/firda-portfolio.git
    ```
2. Navigate to the project directory:
    ```bash
    cd firda-portfolio
    ```
3. Open the `index.html` file in your browser to view the portfolio.
